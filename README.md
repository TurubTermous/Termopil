# Termopil
untuk mengubah sedikit tampilan termux dan menambah kursor kiri kanan
instalasi
$git clone https://gitub.com/TurubTermous/Termopil
$cd Termopil
$bash Termopil.sh

maka lihat lah,tampilan termux akan berubah.
